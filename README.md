# Cancer Signatures in Spaceflight

Manuscript code and processed source tables for the Inspiration4 single-cell cancer-signature analyses in:

**Spaceflight mirrors a pro-carcinogenic cellular signaling matrix**

This repository is the JangKeun Kim code snapshot for the manuscript version reviewed on 2026-06-12. It contains only the processed analysis tables and plotting code needed to regenerate the manuscript-grade Inspiration4 panels and source tables.

## Scope

Included:

- Processed fGSEA result tables for MSigDB Hallmark, C4, and C6 gene sets.
- Processed key cancer gene log2FC table for Inspiration4 PBMC and immune cell subsets.
- R scripts that regenerate manuscript-style panels and selected source tables for Figures 3 and 4.

Not included:

- Raw or sample-level single-cell data.
- Large manuscript image exports.
- Organ-matched rodent and human tumor analyses generated by Josef Borg's workflow.

The raw Inspiration4 data are publicly available through NASA OSDR/SOMA as described in the manuscript. Josef Borg's organ-matched analysis code is maintained separately at `https://github.com/JosefBorg/Spaceflight_vs_Tumor`.

## Repository Layout

```text
data/processed/
  i4_fgsea_hallmark.csv
  i4_fgsea_c4.csv
  i4_fgsea_c6.csv
  i4_key_cancer_genes_log2fc.csv

scripts/
  01_plot_i4_gsea_dotplots.R
  02_plot_key_cancer_heatmaps.R
```

Generated files are written to `outputs/`, which is intentionally ignored by git.

## Reproduce Figures

Run from the repository root:

```bash
Rscript scripts/01_plot_i4_gsea_dotplots.R
Rscript scripts/02_plot_key_cancer_heatmaps.R
```

Expected outputs:

- `outputs/figures/fig4a_i4_c6_top_pathways_dotplot.pdf`
- `outputs/figures/fig4b_i4_c4_top_pathways_dotplot.pdf`
- `outputs/figures/fig4c_i4_hallmark_dotplot.pdf`
- `outputs/figures/fig3b_i4_key_cancer_log2fc_heatmap.pdf`
- `outputs/figures/fig3c_i4_key_cancer_log2fc_r45_r82_heatmap.pdf`
- `outputs/tables/` source and selection tables used by the plotted panels.
